# Raycast Script Commands

## Setup

1. Run `hme setup` at least once to configure your iCloud session cookie
2. Open Raycast Settings (`Cmd+,`) -> Extensions -> Script Commands
3. Click "Add Directories" and select this `raycast/` folder
4. The commands will appear in Raycast search

## Commands

### Generate Hide My Email

- **Keyword**: "Generate Hide My Email"
- **Mode**: compact (result shown inline)
- **Argument**: Label / site name (optional)
- Generates a new address, copies it to your clipboard, and displays it

### Search Hide My Email

- **Keyword**: "Search Hide My Email"
- **Mode**: compact (result shown inline)
- **Argument**: Search query (label or note)
- Finds matching addresses and copies the first match to your clipboard

### List Hide My Email

- **Keyword**: "List Hide My Email"
- **Mode**: fullOutput (scrollable list)
- Shows all existing Hide My Email addresses with labels and forwarding info
